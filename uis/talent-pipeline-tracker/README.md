# Talent Pipeline Tracker

Esta app quedó como contenedor mínimo durante la reorganización del monorepo.

## Estado actual

- Se eliminó la lógica de candidaturas de este frontend para evitar duplicación.
- El módulo de incidents fue movido a backoffice.
- La página principal muestra un mensaje de transición.

## Ejecución

```bash
cd uis/talent-pipeline-tracker
npm install
npm run dev
```

## Nota

Para trabajar con incidents utiliza la app de backoffice en `uis/backoffice`.
